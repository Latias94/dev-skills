# Misc Skills

Misc skills are useful with Codex but are not part of the default Rust engineering workflow.

- **[humanizer](./humanizer/SKILL.md)** — remove signs of AI-generated writing from prose while
  preserving meaning, facts, terminology, and intended voice. Adapted from
  [`blader/humanizer`](https://github.com/blader/humanizer).
- **[logo-generator](./logo-generator/SKILL.md)** — generate SVG logo concepts, PNG exports, and
  polished logo showcase pages or images. Adapted from
  [`op7418/logo-generator-skill`](https://github.com/op7418/logo-generator-skill).
- **[animation-vocabulary](./animation-vocabulary/SKILL.md)** — map loose animation descriptions
  to precise motion terms. Vendored from
  [`emilkowalski/skills`](https://github.com/emilkowalski/skills).
- **[emil-design-eng](./emil-design-eng/SKILL.md)** — apply Emil Kowalski's design engineering
  philosophy to UI polish, component design, and animation decisions. Vendored from
  [`emilkowalski/skills`](https://github.com/emilkowalski/skills).
- **[review-animations](./review-animations/SKILL.md)** — review motion code against strict
  animation standards. Vendored from
  [`emilkowalski/skills`](https://github.com/emilkowalski/skills).
