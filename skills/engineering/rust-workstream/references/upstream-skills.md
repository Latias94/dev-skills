# Upstream Skill Composition

This repo composes with `mattpocock/skills`; it does not vendor those skills.

## Phase Ownership

| Phase | Preferred skill | Output |
| --- | --- | --- |
| Requirement pressure test | `grill-with-docs` | clarified scope, terminology, docs updates |
| Product/spec synthesis | `to-prd` | PRD or issue body |
| Task export | `to-issues` | issue tracker tasks |
| Feature implementation | `tdd` | passing vertical slice |
| Bug/perf investigation | `diagnose` | repro, hypothesis, fix, regression test |
| Architecture review | `improve-codebase-architecture` | refactor candidates |
| Session transfer | `handoff` | compact handoff document |

## Rule

Do not duplicate upstream skill bodies. Reference them by name and decide which phase they own.
